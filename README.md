# morse_decoder
## Built With

- Technologies used:
  > Ruby,
  > Gemfile,
  > IRB


## Getting Started

### Prerequisites

You need to have a browser and this tools installed:

- Git
- NodeJS
- Ruby
- 

### Setup

To get a local copy up and running follow these simple example steps:

```
$ git clone git@github.com:ismailakinkunmi/morse_decoder.git
```

```
$ cd morse_decoder
```

```
$ bundle install
```

### Usage

To run the ruby file:

```
$ ruby (name of file)
```
